# Welcome to the UTA-WESOME!

Members of the UTA-WESOME organization are co-creators of the UTA-GMS system, which allows the use of robustness analysis for preference disaggregation.

## Main projects

- [uta-gms-engine](https://github.com/UTA-WESOME/uta-gms-engine)
- [uta-gms-backend](https://github.com/UTA-WESOME/uta-gms-backend)
- [uta-gms-frontend](https://github.com/UTA-WESOME/uta-gms-frontend)
